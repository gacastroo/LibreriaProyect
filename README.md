# Proyecto Librería

El objetivo de este proyecto es desarrollar un sistema de gestión de una tienda de libros en Java que incluya funcionalidades de manejo de inventario, gestión de clientes, ventas, interfaz gráfica de usuario (GUI) y persistencia de datos en archivos. El proyecto estará dividido en seis módulos, cada uno a cargo de un equipo de programadores, con un equipo adicional encargado de ensamblar los módulos desarrollados.
